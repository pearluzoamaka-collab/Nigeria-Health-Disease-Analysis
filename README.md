# Nigeria-Health-Disease-Analysis
Data-driven health analysis using Power BI to uncover disease trends and mortality patterns.
Project Overview

This project analyzes a Nigerian hospital disease dataset to uncover patterns in disease incidence, mortality rates, and demographic vulnerability using Microsoft Power BI.
The goal was to transform raw healthcare data into actionable insights that can support informed public health decisions.

🎯 Project Objectives

The analysis answers the following key questions:
Which states have the highest incidence of Malaria and Cholera?
Which state has the highest death rate due to Cholera?
Which group has the highest disease occurrence — Rural or Urban?
Which diseases had the highest death rate in 2010 and 2018?
Which year recorded the highest incidence of Meningitis?
Which age group is most susceptible to Diarrhoea?

📊 Key Insights

Malaria remains the most prevalent disease across several states.
Cholera death rates vary significantly between states, indicating disparities in healthcare access and sanitation.
Disease occurrence differs between Rural and Urban settlements.
Mortality patterns shifted between 2010 and 2018.
A peak outbreak year was identified for Meningitis.
Certain age groups are more vulnerable to sanitation-related diseases such as Diarrhoea.

Recommendation

Strengthen malaria prevention in high-incidence states.
Improve sanitation infrastructure in states with high cholera death rates.
Implement targeted health awareness campaigns based on settlement type.
Expand vaccination and early detection programs for outbreak-prone diseases.

Allocate healthcare resources based on data-driven disease burden.
